docker muti-stage build

运维 CI/CD 运维 2019/11/04

前言

docker 多阶段构建是从17.05.0-ce开始支持的，所以你要使用这个功能那么你的docker版本必须大于这个。使用muti-stage build可以减少构建之后的镜像大小，使用依赖缓存，可以在CI/CD的场景中使用，避免管理多个编译环境的尴尬（一台机器你打算装多少nodejs、java版本，此外还有编译工具npm、yarn、maven、gradle😅）
